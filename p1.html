Touch Me if You Can

Virtual Reality Application
C#, Unity, HTC Vive, Leap Motion, Arduino

Overview

This project was designed to enable individuals to engage in social/physical interactions, such as shaking hands, even if they are not physically close to one another.

To achieve this, the project used virtual reality with an internet connection to join users into one virtual space. To further enhance user experience, the project also incorporated Leap Motion, a tracking device that converts users’ hand motions into virtual reality, and Arduino vibrators, which generate physical pulses to imitate haptic feedback. Using C# language powered by Unity, I was responsible for the integration of these components into a desirable and product-like system, complete with gloves and clothes equipped with vibrators signal physical feedback on different parts of body. With this system, users can not only interact with each other remotely with greater immediacy, but can also literally “feel” each other’s actions, like handshakes or pats on the shoulder, by getting haptic feedback via the technology.

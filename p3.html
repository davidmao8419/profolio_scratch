AR Football Game

Augumented Reality Application
C#, Unity, Vuforia

Overview

This is a football shooting game combined with augumented reality.
Once the player finds the front side of Eclipse Mints box, the game will begin. The goal and the football will appear as soon as the box is detected. Player needs to use the mouse to control the ball. If the ball hits the moving obstacle in front of the goal, player will lose score.

It was developed with Vuforia, an AR platform, and Unity with C# language.

This project won third price of Best Final Project from Introduction to Multimedia class in National Tsing Hua University.

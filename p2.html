Team Formation

Web Application
Angular 2, Firebase, Amazon Turk, Natural Language Processing

Overview

This project investigate factors related to team formation, and in particular, whether we can leverage individuals' proposed solutions to a problem as a way to form effective teams.

Using Angular 2, I built a web system from scratch for participants to conduct experiments on Amazon Turk. I used Firebase developed by Google as the database.


The work flow of the experiment system:
Each individual first entering the system would be asked for several quesitons about themselves, such as personality.
These information would be stored to assist future research.
We then randomly assigned participants into groups.
Each group was assigned a designated problem and team members communicated via online messaging in the system to determine a solution of the specific problem.
To determine if higher homogeneity among teammates leads to a better overall conclusion, meaning a better team, all chat history and individual information were stored in JSON form in Firebase and analyzed using Topic Model with Python to glean a deeper insight into the unstructured bodies of text and visualize the connection of each individual teammate with the team conclusion.
